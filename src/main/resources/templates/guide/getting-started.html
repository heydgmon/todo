01 GETTING STARTED

30초 만에 시작하기

do·it 서비스를 처음 사용하는 분을 위한 빠른 시작 가이드입니다. 회원가입 없이 바로 시작하세요.

설치 없이 바로 시작

do·it은 별도의 앱 설치나 회원가입 없이 웹 브라우저만으로 바로 사용할 수 있는 생산성 도구입니다. PC, 태블릿, 스마트폰 어디서든 접속하면 됩니다.

1

사이트 접속

taskall.click에 접속하면 메인 페이지가 나타납니다. 깔끔한 랜딩 페이지에서 서비스의 핵심 기능을 미리 확인할 수 있습니다.

2

바로 시작하기 클릭

"시작하기" 또는 "대시보드 열기" 버튼을 누르면 곧바로 게스트 모드로 진입합니다. 회원가입이나 로그인 없이도 할 일 등록, 캘린더 관리, 대시보드 확인이 모두 가능합니다. 게스트 모드의 데이터는 브라우저의 LocalStorage에 저장되므로, 같은 브라우저에서는 브라우저를 닫았다 열어도 데이터가 유지됩니다.

3

필요할 때 로그인

데이터를 클라우드에 저장하거나, 다른 기기에서도 이어서 쓰고 싶다면 상단의 "로그인" 버튼으로 가입하세요. Amazon Cognito를 통한 안전한 OAuth2 인증 방식으로, 이메일 주소만으로 간편하게 가입할 수 있습니다.

🖥️ 사이드바 네비게이션

로그인하거나 게스트 모드로 진입하면, 화면 왼쪽에 사이드바가 나타납니다. 사이드바에서 대시보드, 캘린더, 할 일 관리, 워크스페이스, 설정 페이지로 바로 이동할 수 있습니다. 화면 오른쪽 하단의 ? 버튼을 누르면 언제든 이 사용 가이드로 돌아올 수 있습니다.

💡 Tip

처음 사용한다면 먼저 게스트 모드로 기능을 충분히 체험해보고, 마음에 들면 로그인해서 데이터를 클라우드에 영구 보관하는 것을 추천합니다.
